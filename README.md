# 📊 Retail Sales Performance Dashboard (Power BI)

## 🔍 Project Overview
This project analyzes retail sales data to identify key trends, top-performing products, and regional performance using Power BI.

The goal is to help businesses make data-driven decisions by understanding sales patterns and performance.

---

## 📈 Key Insights
- West region contributes the highest sales  
- Tablets and electronics generate the most revenue  
- Sales trends fluctuate over time indicating demand patterns  

---
